The Language of Space

le modèle

Maxim Sokhatsky20 DEC 2018

Here is the conceptual model of a theorem-proving system that unifies tower of type systems with runtime components (as extraction targets). The model has presented here as a set of 1) calculi with its models; 2) programs and base libraries; and 3) program transformations: type checking, compilation, type erasure, code extraction, optimization, etc.

While being a generic blueprint, some parts of the model implemented in various languages. The research was founded by Synrc Research Center, Kyiv, Ukraine. All works were done under the supervision of Pavlo Maslianko, National Technical University of Ukraine «Igor Sikorsky Kyiv Polytechnic Institute» Faculty of Applied Mathematics.

la genèse

The process of formal verification has following phases: 1) model encoding, 2) linking with base library of math components, 3) desugaring to core functions, 3) linking with pure base library, and 4) extracting.

Minimal Model. This model can serve PTS (Pure Type System) research, being a CoC extension (Calculus of Construction) system with infinite number of universes with extraction of pure programs to certified interpreter.



Cubical Model. This model can serve HoTT research, being MLTT/CCHM (Martin-Löf Type Theory, Cohen-Coquand-Huber-Mörtberg) system with limited extraction of pure programs to certified interpreter.


Spectral Model. This model intended to server needs of linkage with $\pi$-calculus runtimes and optionally stream calculuses and fusion engines. Extraction to Erlang or CPS (Continuation Passing Style) interpreter.
